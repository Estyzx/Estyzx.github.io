# CSS样式

CSS，专门用来“美化”标签


## CSS应用方式
1. 在标签上
```html
<img src = "..." style = "..."/>
```
2. 在head标签中写style标签
```html
<head>
<style>
    .c1{
        color: red;
    }
</style>
</head>
<body>
    <h1 class = "c1"></h1>
</body>
```
3. 在外部文件
```html
<head>
    <link rel = "stylesheet" href = "css/style.css">
</head>
<body>
    <h1 class = "c1"></h1>
</body>
